## Starting Your First Project

Behruz Toshpulatov provides a space to organize project docs easily. Follow these steps:

1. Log in to your account at `https://portal.behruztoshpulatov.com`.
2. Click "Create New Project" on the dashboard.
3. Name your project and add a short description.
4. Begin adding pages by clicking "Add Page."
5. Use Markdown or the editor to write and organize your documentation.

*Pro Tip:* Use folders to keep related docs grouped together.
