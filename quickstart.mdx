## Prerequisites

Before starting, ensure you have the following:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: Node.js `{>18.0.0}` for CLI tools

</Callout>

## Create Your Account

Sign up for Behruz Toshpulatov to access your documentation workspace.

<Tabs>
  <Tab title="Web Browser" icon="globe">

Visit `https://dashboard.example.com/signup` and follow these steps:

<Steps>
  <Step title="Enter Details" icon="user">
    Provide your email, name, and a strong password.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox and click the verification link.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to `https://dashboard.example.com/login` and sign in.
  </Step>
</Steps>

  </Tab>
  <Tab title="API" icon="code">

Use the API to create an account programmatically.

<Request tabs="cURL,JavaScript" show-lines="true">
```bash
curl -X POST https://api.example.com/v1/accounts \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "name": "Behruz Toshpulatov",
    "password": "your-secure-password"
  }'
```
```javascript
const response = await fetch('https://api.example.com/v1/accounts', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'user@example.com',
    name: 'Behruz Toshpulatov',
    password: 'your-secure-password'
  })
});
```
</Request>

  </Tab>
</Tabs>

## Create Your First Documentation Space

Once logged in, set up a space for your project docs.

<Steps>
  <Step title="Navigate to Spaces" icon="book-open">
    Click the `{+}` icon in the sidebar or go to Spaces dashboard.
  </Step>
  <Step title="New Space" icon="plus">
    Enter a name like "My Project Docs" and select a template.
  </Step>
  <Step title="Configure" icon="settings">
    Set visibility to private or public, then save.
  </Step>
</Steps>

<Response tabs="201" show-lines="false">
```json
{
  "id": "space_abc123",
  "name": "My Project Docs",
  "url": "https://dashboard.example.com/spaces/space_abc123"
}
```
</Response>

## Upload Initial Project Files

Add your markdown files or assets to the space.

<CodeGroup tabs="Drag & Drop,cURL API">
````bash
# Drag files directly to the editor upload zone
# Supports .md, .png, .json up to 10MB
````

````bash
curl -X POST https://api.example.com/v1/spaces/space_abc123/files \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@README.md"
````
</CodeGroup>
